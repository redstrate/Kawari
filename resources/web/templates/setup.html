Connecting to this server with Astra is very easy, as the functionality is built-in.

  1. Open Astra, tap the "Settings" button in the top-right.
  2. Under the General page, check the "Show Developer Settings" checkbox.
  3. Under the Accounts page, select the account you created.
  4. Select the "Developer" tab.
  5. Use the following URL as the Server URL: {{ server_url }}.
  6. Select "Download Configuration".

Astra will automatically download the correct server configuration. That's it!

It's possible to manually run the game if you can't or don't want to use a launcher.

Since the command line arguments include a unique SID, you must login first. Then, go to the Login History page and follow the instructions there.
